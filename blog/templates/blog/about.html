About this project

This is a small portfolio project created by Brian Olsen (Username: brianolsen). This project is a small reproduction of the popular social networking website, Twitter. Unlike the current Twitter, this reproduction focuses on the essence of what made the website good, the need to condense thoughts down to 140 characters. In this implementation, I have brought back the original character limit and removed the media sharing functionality. The end result? Straight to the point text based social networking.

How is it made?

This project is built primarily using Django, a popular Python framework. The website design was created using Bootstrap classes. Currently, the website uses a local SQLite database to store the User and Post data. I am planning to upgrade the database to MySQL or PostgreSQL if this project is made active online.
